DouZero是一个为斗地主设计的强化学习框架。斗地主十分具有挑战性。它包含合作、竞争、非完全信息、庞大的状态空间。斗地主也有非常大的状态空间，并且每一步合法的牌型会非常不一样。DouZero由快手AI平台部开发。
